## 🐍 Python часть

### Используемые библиотеки
```python
import pandas as pd
import numpy as np
```

Демонстрируемые параметры read_excel()
```
Параметр	Описание	Пример использования
sheet_name	Выбор листа	sheet_name='Предметы', sheet_name=2
skiprows	Пропуск строк	skiprows=1
header	Заголовки	header=0, header=None
names	Свои названия столбцов	names=['ID', 'Имя']
usecols	Выбор столбцов	usecols='A:C', usecols=[0,1]
nrows	Количество строк	nrows=3
dtype	Типы данных	dtype={'ID': str}
sheet_name=None	Все листы сразу	sheet_name=None
```

Фрагмент кода (Python)
```
# Чтение нескольких листов одновременно
sheets = pd.read_excel("data.xlsx", 
                       sheet_name=['Студенты', 'Предметы'],
                       skiprows=0)

# Выборочное чтение столбцов
df = pd.read_excel("data.xlsx", 
                   sheet_name="Студенты",
                   usecols=['Имя', 'Средний балл'],
                   nrows=3)
```

### 🎯 C# часть

Используемые пакеты
```
<PackageReference Include="ExcelDataReader" Version="3.6.0" />
<PackageReference Include="ExcelDataReader.DataSet" Version="3.6.0" />
<PackageReference Include="System.Text.Encoding.CodePages" Version="8.0.0" />
```
Функциональность

1. Последовательное чтение всех листов
2. Чтение через DataSet (удобная работа с таблицами)
3. Выборочное чтение (эмуляция usecols и nrows)
4. Явное преобразование типов (эмуляция dtype)

Фрагмент кода (C#)

```
// Регистрация кодировки для русского текста
Encoding.RegisterProvider(CodePagesEncodingProvider.Instance);

// Чтение Excel файла
using (var stream = File.Open("data.xlsx", FileMode.Open, FileAccess.Read))
using (var reader = ExcelReaderFactory.CreateReader(stream))
{
    // Чтение всех листов
    var result = reader.AsDataSet();
    
    // Работа с конкретным листом
    var studentsTable = result.Tables["Студенты"];
}
```
